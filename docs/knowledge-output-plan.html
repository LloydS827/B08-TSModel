知识成果规划

B08 的成果不只是一段模型代码。由于课题目标是设备时序基础模型，应同步规划专利、论文、数据集、benchmark 和模型资产。

专利方向

  • 阶段感知多物理域设备时序基础模型。
  • 基于仿真退化数据的设备预损坏表征学习方法。
  • 无故障标签条件下的设备损坏前兆评测方法。
  • 工艺阶段 token 与传感器物理域 token 的联合建模方法。

论文方向

  • Industrial Small-Sample Time-Series Foundation Model。
  • Simulation-Driven Pretraining for Predictive Maintenance。
  • Process-Stage-Conditioned Multivariate Time-Series Representation Learning。

数据与评测

  • 真空速凝炉模拟数据集。
  • 阶段化窗口数据集。
  • 退化注入 benchmark。
  • 开源模型适配评测榜单。

模型资产

  • 模型 IO schema。
  • 开源模型 adapter。
  • 轻量微调权重。
  • 小型设备时序基础模型原型。
