证据地图

从来源到下一步实验

1
E1 forecasting residual

用 TTM / Chronos / TimesFM / Moirai 等 forecasting-first 模型形成残差候选信号，但不解释为生产告警。

2
E2 representation

用 MOMENT / UniTS 等表征能力验证阶段、质量或弱标签 proxy 的可分性，不解释为真实健康等级。

3
E3 imputation

用 mask reconstruction / imputation 验证多传感器缺失与重建任务，不解释为异常识别。

4
E4 open data PM

用 C-MAPSS、IMS、PRONOSTIA、TEP 补充故障、退化、RUL 或过程异常证据，不外推为 FU13 已具备 RUL。

5
E5 patent effect

为 P1-P5 形成技术效果样例、对照表和 prior-art 差异材料，不解释为专利授权结论。

推荐阅读顺序

先判断，再深入

  1. 先读 论文与专利方向，明确主线和下一步证据需求。
  2. 再读 来源登记表，检查每个结论对应的 source ID。
  3. 需要学术背景时读 学术主线综述 和 模型矩阵。
  4. 进入实验设计前读 任务与指标矩阵 和 数据集矩阵。
  5. 只有 C 阶段证据充分后，再读 训练路线 判断 B 阶段是否启动。
边界提醒

当前不主张的能力

当前 HTML 索引用于阅读和决策，不改变研究结论边界：不承诺生产告警、FU13 RUL 精确估计、自动维修建议、专利授权或自研基础模型训练成功。所有 C/B 阶段工作仍需后续实验、专家复核和数据授权支撑。
