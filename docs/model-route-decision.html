模型路线决策

B08 的关键决策不是“要不要做模型”，而是先用开源、微调，还是自训。决策依据来自同一套模拟数据和真实片段评测。

路线对比

路线进入条件退出条件产出知识价值
直接引用zero-shot 预测和 embedding 任务稳定优于 baseline无法处理阶段语义或退化前兆adapter、评测报告、工程引用建议较低
轻量微调开源模型有效但领域偏差明显微调收益低或部署复杂度过高阶段/传感器 adapter、微调权重、领域评测中高，适合专利和应用论文
领域自训开源模型不满足输入输出和任务需求数据规模不足以支撑稳定预训练小型设备时序 foundation model、预训练任务、benchmark最高，但风险最大

推荐节奏

阶段动作目标
第 1 阶段直接引用 + frozen embedding快速判断已有模型是否可用
第 2 阶段轻量微调验证阶段 token 和物理域适配是否带来增益
第 3 阶段领域自训可行性只有当开源路线不足时启动
